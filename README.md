# 📚 Course Manager

## 🚀 Live Demo
🔗 Live Site: https://noorkaram.github.io/CoursesManager-crud-app/

---

## 🛠️ Tech Stack
- HTML5 – Semantic structure
- CSS3 – Custom styling & responsive layout
- JavaScript (ES6+) – OOP, Fetch API, async/await
- Bootstrap 5 – UI components & grid system
- SweetAlert2 – Alerts & confirmation dialogs
- MockAPI – Cloud-hosted REST API
- Git & GitHub – Version control

---

## ✨ Features
- 📋 View all courses in a clean table layout
- ➕ Add new courses with a validated form
- ✏️ Edit existing courses with a pre-filled form
- 🗑️ Delete courses with a confirmation dialog
- ✅ Form validation (all fields required, min price 1000 LE)
- 📱 Fully responsive on desktop and mobile
